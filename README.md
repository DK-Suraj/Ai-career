# 🚀 AI Career Guidance System

An intelligent **AI-powered Career Guidance Web Application** built using **Spring Boot**.  
This platform helps users analyze their skills, evaluate resumes, and get AI-based career suggestions.

---

## 🌟 Features

- 🔐 User Registration & Login (with Email OTP Verification)
- 🔑 Forgot Password with OTP Reset
- 👤 Profile Management (Photo Upload, Skills, Education)
- 🧠 AI Skill Analyzer
- 📄 Resume Analyzer (Upload PDF)
- 🤖 AI Chat Assistant (Career Questions)
- 📊 Dashboard with User Data
- 📱 Responsive UI (Bootstrap)

---
